# Data Engineering Training (VNR 2024-2028)

This is the code repository of the project based industry-aligned training program for the **Data Engineer** roles.

In building these Data powered products, students exercised their skills across various tools of data engineering by working in a collaborative setting of Agile squad/teams.

This program is run in collaboration with **VNR VJIET** (https://vnrvjiet.ac.in/). 

If you would like to be part of similar future training programs, please drop me an email at `kvamsi82@gmail.com`.
